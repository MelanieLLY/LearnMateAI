## Project 3: Team Application with Advanced AI 

**Due:** Apr 19 by 11:59pm | **Points:** 200 | **Weight:** 20% of final grade

### Author
老师的官方要求

### Objective
Build a production-grade application with advanced AI techniques as a team.

### Approval Requirement
Project idea must be approved by the professor on the #projects Slack channel at least one week before the deadline.

### Functional Requirements
- **Complex, production-ready application**
- **Multiple user types** with different capabilities
- **Advanced features** requiring sophisticated logic
- **Real-world use case** (Mom Test validated)
- **Portfolio/interview-worthy quality**

### Technical Requirements
- **Advanced Architecture:** Microservices or well-architected monolith, caching strategy, API gateway.
- **Parallel Agentic Programming:** 3+ features built with parallel agents, documented workflow.
- **Comprehensive Evaluation System:** Multi-dimensional evals, LLM-as-judge, historical metrics.
- **Enterprise CI/CD:** Multi-environment, canary/blue-green deployment, performance gates.
- **Production Monitoring:** Error tracking, APM, uptime monitoring, alerting.
- **Security:** Security audit, OWASP top 10 addressed, secrets management.

### Team Process
- **3+ sprints** documented.
- **Team contract** established.
- **Code review** process.
- **Peer evaluations**.

### Deliverables
- GitHub organization with repositories.
- Deployed application (production).
- Monitoring dashboard access.
- Eval system with historical data.
- Complete documentation package.
- 20-minute team presentation.
- Individual reflections.
- Technical blog post.
